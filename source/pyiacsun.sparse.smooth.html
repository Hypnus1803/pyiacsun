pyiacsun.sparse.smooth.normSquared.normSquared(x, A, At, b=0, c=0, constant=0)¶

f, g = normSquared(x,A,At,b,c,constant) returns the objective function f and its gradient

A is a function that computes the matrix-vector product At is a function that computes the matrix-vector product for the transposed matrix

By default, b=0 and c=0
